# 중국어 단어 시험 오답노트 웹서비스

> 초급 중국어 학습자가 직접 단어를 등록하고, 자동 시험과 오답 복습을 통해 단어를 반복 학습할 수 있도록 만든 웹서비스입니다.

## 무엇을 하는 프로젝트인가

이 프로젝트는 중국어를 처음 배우는 학습자를 위한 단어 시험 및 오답노트 웹서비스입니다.

사용자는 중국어 단어, 병음, 한국어 뜻을 직접 입력할 수 있습니다.
입력한 단어를 바탕으로 웹사이트에서 자동으로 시험을 볼 수 있고, 시험에서 틀린 단어는 오답노트에 저장됩니다.
사용자는 오답노트에 저장된 단어만 다시 시험 보면서 자신이 자주 틀리는 단어를 반복해서 복습할 수 있습니다.

이 프로젝트를 만든 이유는 중국어 단어 학습에서 한자, 병음, 뜻을 함께 확인해야 하기 때문입니다.
종이 단어장으로 공부하면 어떤 단어를 반복적으로 틀리는지 확인하기 어렵지만, 이 웹서비스를 사용하면 오답이 자동으로 모여 학습자가 자신의 부족한 부분을 더 쉽게 확인할 수 있습니다.

## 주요 기능

- 중국어 단어, 병음, 한국어 뜻 등록
- 등록한 단어 목록 확인
- 뜻을 보고 중국어 단어와 병음을 입력하는 자동 시험
- 정답 / 오답 판별
- 틀린 단어 오답노트 저장
- 오답 단어 재시험
- 전체 단어 수, 정답 수, 오답 수, 정답률 확인

## 사용 방법

1. 단어 등록 화면에서 중국어 단어, 병음, 한국어 뜻을 입력합니다.
2. 단어 추가 버튼을 눌러 단어를 저장합니다.
3. 단어 목록 화면에서 등록한 단어를 확인합니다.
4. 자동 시험 화면에서 시험을 시작합니다.
5. 화면에 나온 한국어 뜻을 보고 중국어 단어와 병음을 입력합니다.
6. 제출 버튼을 누르면 정답 또는 오답 결과가 표시됩니다.
7. 틀린 단어는 오답노트에 저장됩니다.
8. 오답 재시험 버튼을 눌러 틀린 단어만 다시 복습할 수 있습니다.

## 사용 예시

예시로 다음과 같은 단어를 등록할 수 있습니다.

| 중국어 단어 | 병음 | 한국어 뜻 |
|---|---|---|
| 学习 | xuéxí | 공부하다 |
| 老师 | lǎoshī | 선생님 |
| 图书馆 | túshūguǎn | 도서관 |
시험 화면에서 "공부하다"가 문제로 나오면 사용자는 다음과 같이 입력합니다.

| 입력 항목 | 사용자가 입력하는 답 |
|---|---|
| 중국어 단어 | 学习 |
| 병음 | xuéxí |

중국어 단어와 병음이 모두 맞으면 정답으로 처리됩니다.
둘 중 하나라도 틀리면 오답으로 처리되고, 해당 단어가 오답노트에 저장됩니다.

## 입력 / 출력

| 입력 | 출력 |
|---|---|
| 중국어 단어 | 단어 목록에 저장 |
| 병음 | 시험 정답 판별 기준으로 사용 |
| 한국어 뜻 | 시험 문제로 제시 |
| 사용자의 시험 답안 | 정답 / 오답 결과 표시 |
| 오답 처리된 단어 | 오답노트에 저장 |
| 시험 기록 | 정답 수, 오답 수, 정답률로 표시 |

## 아직 안 되는 것 / 보완할 점

- 현재는 사용자가 직접 입력한 단어를 기준으로만 시험을 볼 수 있습니다.
- 단어 데이터를 외부 파일로 불러오는 기능은 아직 없습니다.
- 회원가입이나 로그인 기능은 없습니다.
- 여러 기기에서 학습 기록을 공유하는 기능은 없습니다.
- 중국어 입력기 설정이 필요해서 초보자에게는 어려울 수 있습니다. 
- 디자인과 화면 구성은 더 단순하고 보기 좋게 개선할 수 있습니다.
- 병음의 성조 표기를 사용자가 다르게 입력했을 때 어느 정도까지 정답으로 인정할지 추가 기준이 필요합니다.

## 프로젝트를 만들며 확인한 점

이 프로젝트를 만들면서 코드 자체를 모두 직접 이해하기보다는, 사용자의 입장에서 실제로 기능이 작동하는지 확인하는 방식으로 검증했습니다.

PRD에 적은 기능이 실제 웹사이트에서 구현되었는지 하나씩 비교했고, 빈칸 입력이나 오답 입력처럼 문제가 생길 수 있는 상황도 확인하려고 했습니다.

이를 통해 AI가 만든 결과물이라도 사용자가 직접 실행하고 검증해야 프로젝트의 완성도를 판단할 수 있다는 점을 확인했습니다.
